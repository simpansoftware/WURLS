# WURLS - W URL Shortener
## Demo
Demo is hosted [right here](https://wurls.4lvaret.tech/).
## What is it?
It's an URL shortener, with some pretty nifty features (please dont use this commercially read main.py to understand why)
## Features
- It shortens urls
- Generates qr codes for urls
- No JS needed!
- URL validation!
## Usage
### To go to a shortened website 
Go to https://wurls.4lvaret.tech/l/?url= and then whatever your shortened ID is to go to your shortened webpage<br>
Example: Go to [https://wurls.4lvaret.tech/l/?url=NZPDI](https://wurls.4lvaret.tech/l/?url=NZPDI) and that will bring you to Google
### To see the URL for a shortened website
Go to https://wurls.4lvaret.tech/g/?url= and then whatever your URL is to get the shortened link for your URL<br>
Example: Go to [https://wurls.4lvaret.tech/g?url=https://google.com](https://wurls.4lvaret.tech/g?url=https://google.com) and that will show you the shortened link of Google
### To get a QR code of a shortened website
Go to https://wurls.4lvaret.tech/qr/?url= and then whatever your shortened ID is to get the QR code for your URL<br>
Example: Go to [https://wurls.4lvaret.tech/qr/?url=NZPDI](https://wurls.4lvaret.tech/qr/?url=NZPDI) and that will show you the QR code for Google
